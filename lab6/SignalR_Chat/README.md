# Система интерактивного взаимодействия (SignalR)
**Разработчик:** Даниил Белов
**Дисциплина:** Информационные системы

## Описание модуля
Данный программный модуль реализует систему обмена данными в реальном времени на базе технологии **SignalR**. 
В рамках лабораторной работы №6 были решены следующие задачи:
1. Настройка серверного хаба для управления потоками сообщений.
2. Реализация клиентской логики на JavaScript для мгновенного обновления интерфейса.
3. Интеграция системы уведомлений о подключении/отключении пользователей.

## Стек технологий
* **Backend:** ASP.NET Core 8.0, SignalR Hubs
* **Frontend:** JavaScript (SignalR Client), HTML5, CSS3 (Custom Design)

## Инструкция по развертыванию
1. Скомпилируйте проект через Visual Studio или CLI: `dotnet build`.
2. Убедитесь, что клиентская библиотека SignalR находится по пути `wwwroot/js/signalr/`.
3. Запустите приложение. Чат будет доступен по адресу `/Chat`.
