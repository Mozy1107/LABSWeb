## Описание проекта
В данном проекте реализованы три независимые информационные подсистемы, объединенные общим интерфейсом и современным дизайном. Каждая подсистема демонстрирует работу с объектно-ориентированными моделями данных, базами данных (Entity Framework Core) и механизмами фильтрации/запросов.

---

## Реализованные модули (Варианты)

### 1. Система управления образовательным процессом (Вариант 1)
Информационная модель для учета академического состава и обучающихся.
* **Функционал:** Учет абитуриентов, студентов и преподавателей.
* **Ключевые запросы:** Выборка студентов по номеру курса, поиск преподавателей по кафедрам.

### 2. Система управления кадрами предприятия (Вариант 3)
Модуль для автоматизации отдела кадров промышленного предприятия.
* **Функционал:** Разграничение ролей между рабочими, инженерным составом и администрацией.
* **Ключевые запросы:** Поиск рабочих по цехам, статистика инженеров в подразделениях.

### 3. Библиотечный фонд и издательство (Вариант 5)
Система каталогизации печатных изданий.
* **Функционал:** Работа с книгами, журналами и учебными пособиями.
* **Ключевые запросы:** Фильтрация изданий по году выпуска, расчет общего количества учебного материала.

---

## Дополнительные технологии
* **SignalR:** В проект интегрирован интерактивный чат для взаимодействия пользователей в реальном времени.
* **UI/UX:** Применен кастомный дизайн с акцентом на читабельность и профессиональный стиль (Inter Font, Soft Blue Theme).
* **EF Core:** Использование подхода Code First для генерации структуры таблиц.

## Инструкция по запуску
1. Выполните команду `dotnet restore` для загрузки зависимостей.
2. Примените миграции: `dotnet ef database update`.
3. Запустите приложение через Visual Studio или командой `dotnet run`.
